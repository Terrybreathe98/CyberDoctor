# 痰湿体质与高血脂风险关键指标筛选分析报告

## 一、 分析概述

- **样本量**：1000 例
- **高血脂患病率**：79.3% (793/1000)
- **痰湿质评分分布**：均值 32.98 ± 20.13，范围 [0, 65]
- **分析目标**：结合相关性、回归分析、机器学习特征重要性，筛选与痰湿体质及高血脂风险关联的关键生物标志物与活动能力指标。

## 二、 痰湿质评分相关性分析

### 2.1 Pearson 相关性结果

痰湿质评分与各候选指标的线性相关性整体较弱，绝对值均小于 0.1。唯一具有统计学显著性（p < 0.05）的指标为 **ADL吃饭**（r = -0.075）。

| 指标 | 相关系数 r | p 值 | 显著性 |
| :--- | :--- | :--- | :--- |
| ADL吃饭 | -0.075 | 0.018 | * |
| TC（总胆固醇） | -0.042 | 0.186 | ns |
| ADL总分 | -0.039 | 0.218 | ns |
| ...（其他指标略）| ... | ... | ... |

**结论**：痰湿质评分与血常规指标及日常活动能力的直接线性关联不显著，需结合非线性或多因素模型进一步挖掘。

### 2.2 多元线性回归与 LASSO 回归

- **逐步回归（向前选择）**：仅 **ADL吃饭** 被纳入最终模型（R² = 0.006，p = 0.018）。
- **LASSO 回归**：同样仅筛选出 **ADL吃饭**（系数 = -0.582），最优 Alpha 值为 0.924。

**小结**：在线性假设下，仅 "ADL吃饭能力" 对痰湿质评分有微弱的负向预测作用。

## 三、 高血脂风险预测分析

### 3.1 Logistic 回归结果

- **单因素分析显著指标（p < 0.05）**：
  - **强危险因素**：TG（甘油三酯）[OR=7.82]、LDL-C [OR=2.22]、TC [OR=2.08]、血尿酸 [OR=1.003]。
  - **保护因素**：HDL-C [OR=0.175]。

- **多因素 Logistic 回归独立预测因子**：
  - TC（总胆固醇）、TG（甘油三酯）、LDL-C（低密度脂蛋白）、血尿酸（正向关联）。
  - HDL-C（高密度脂蛋白）（负向关联）。
  - *注：模型可能存在准完全分离现象（23%样本可被完美预测），提示上述血指标对高血脂判别极强。*

### 3.2 ROC 曲线分析

评估各指标对高血脂状态的单独区分能力：

| 指标 | AUC 值 | 评估 |
| :--- | :--- | :--- |
| **TG（甘油三酯）** | **0.848** | 优秀 |
| **TC（总胆固醇）** | **0.810** | 良好 |
| **血尿酸** | **0.646** | 尚可 |
| **LDL-C（低密度脂蛋白）** | **0.631** | 尚可 |
| HDL-C（高密度脂蛋白） | 0.382 | 反向指示 |
| 其他活动量表指标 | ~0.50 | 无区分能力 |

**结论**：血生化指标（尤其 TG 和 TC）对高血脂状态具有极高的识别能力，而日常生活活动能力指标（ADL/IADL）无明显预测价值。

## 六、 结论

1. **高血脂预测核心因子**：血脂四项（TG, TC, LDL-C, HDL-C）及血尿酸是识别高血脂风险的**决定性生物标志物**，其 AUC 和 Logistic 回归系数均表现优异。
2. **痰湿体质关联特征**：痰湿质评分与临床指标的线性关联较弱。仅 **ADL吃饭能力** 显示出微弱的负相关性，提示**饮食自主性差可能与痰湿体质倾向有关**。
3. **活动量表指标价值**：IADL/ADL 量表在预测高血脂硬终点（患病率）方面贡献极小，但在解释痰湿体质（中医症候群）内部变异时具有一定的非线性和边缘线性参考意义。
